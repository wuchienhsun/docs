Welcome to the **Sending SaaS** developer documentation. Our platform provides a unified API for multi-channel messaging, allowing you to easily send emails and SMS through a smart routing system that optimizes for cost and deliverability.

## Key Features

- **Unified API**: Single interface for multiple providers (Twilio, SendGrid, AWS SES, etc.).
- **Smart Routing**: Automatically routes traffic to the most cost-effective provider for a given region.
- **Failover**: Automatic retry on backup providers if the primary fails.
- **Security**: Granular API keys and secure authentication.

## Getting Started

To get started with the Sending SaaS API, you'll need an API key.

1.  **[Sign up](https://dashboard.sending-saas.com)** for an account.
2.  Navigate to the **API Keys** section in your dashboard.
3.  Create a new API Key.
4.  Check out the [Quickstart](/quickstart) guide to send your first message.

## Base URL

All API requests should be made to:

```bash
https://api.sending-saas.com/api/v1
```
