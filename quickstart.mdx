## 1. Get your API Key

Login to the [Dashboard](https://change.me), navigate to **API Keys**, and create a new key.

## 2. Send an Email

Use the `curl` command to send your first email. Replace `<your_api_key>` with the key you generated.

```bash
curl -X POST https://api.sending-saas.com/api/v1/send/email \
  -H "Authorization: Bearer <your_api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "from": "onboarding@your-domain.com",
    "to": ["delivered@resend.dev"],
    "subject": "Hello World",
    "html_body": "<strong>It works!</strong>"
  }'
```

## 3. Verify Delivery

Check the **Messages** tab in your dashboard to see the status of your email.
