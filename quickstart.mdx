This guide will help you send your first Email and SMS using the Sending SaaS API.

## Prerequisites

- An active [Sending SaaS account](https://dashboard.sending-saas.com).
- A valid JWT Token (obtainable via authentication endpoints).

## Authentication

Authenticate your requests by including your JWT Token in the `Authorization` header.

```bash
Authorization: Bearer <your_jwt_token>
```

## 1. Send an Email

Use the `/send/email` endpoint to send a transactional email.

```bash
curl -X POST https://api.sending-saas.com/api/v1/send/email \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "to": "user@example.com",
    "subject": "Hello from Sending SaaS",
    "html": "<strong>Welcome!</strong> This is your first email."
  }'
```

## 2. Send an SMS

Use the `/send/sms` endpoint to send a text message.

```bash
curl -X POST https://api.sending-saas.com/api/v1/send/sms \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "to": "+1234567890",
    "body": "Your verification code is 123456"
  }'
```
